## 1. 品牌与设计系统

### 1.1 品牌定位

> **FMT微生态 · GAP营养小屋** — AI 驱动的个性化精准营养平台

品牌核心：以肠道微生态（FMT）为科学基石，通过自研知识库与 AI 匹配模型，为不同人群提供个性化营养方案。

### 1.2 配色系统

| Token | Hex | 色值 | 用途 |
|---|---|---|---|
| `--bg` | `#FDFBF7` | 暖白米色 | 所有页面背景 |
| `--card-bg` | `#FFFFFF` | 白色 | 卡片、表单、弹窗 |
| `--primary` | `#6B705C` | 鼠尾草绿 | CTA 按钮、选中导航、标题强调 |
| `--secondary` | `#B2B8A3` | 苔藓绿 | 次要元素、图表区、统计卡片 |
| `--accent` | `#D4A373` | 陶土色 | Footer、Admin 导航选中、渐变色 |
| `--border` | `#E8D5B7` | 沙色 | 卡片边框、分割线 |
| `--text-primary` | `#2C2C2C` | 深灰 | 主标题、正文 |
| `--text-secondary` | `#5C5C5C` | 中灰 | 辅助文本 |
| `--surface-alt` | `#F5F2EB` | 浅米 | Tech 区背景、侧栏、备用卡片 |

### 1.3 字体与排版

- 字体系列：系统默认无衬线（Inter / SF Pro）
- 标题：`font-bold` + 大字号（`text-3xl` ~ `text-5xl`）
- 英文标签：`tracking-[0.15em] uppercase` + `font-light` / `font-semibold`
- 正文：`text-sm` / `text-base` + `leading-relaxed`

### 1.4 圆角与阴影

- 卡片圆角：`rounded-[20px]` ~ `rounded-[32px]`
- 按钮圆角：`rounded-full`（胶囊形）
- 卡片阴影：`box-shadow: 0 4px 24px rgba(108, 112, 92, 0.04~0.06)`

### 1.5 图标系统

使用 [Lucide React](https://lucide.dev/icons) 图标库。产品分类对应图标：

| 分类 | icon | 分类 | icon |
|---|---|---|---|
| growth | Sprout | senior | Shield |
| vitality | Zap | recovery | Dumbbell |
| balance | Heart | gut | Leaf |
| energy | Activity | immune | Brain |
| maternal | Baby | sleep | Moon |

## 3. Landing 页面详解

### 页面结构（纵向瀑布流）

```
Navbar（固定顶部）
  ├── Logo: FMT微生态 / GAP营养小屋
  ├── 导航: 产品 | 技术 | 知识库
  └── CTA: 加入会员

Hero（全屏首屏）
  ├── 背景: 浮动渐变圆（B2B8A3 / D4A373 / E8D5B7）
  ├── 主标题: "精准营养 · 科学驱动"
  ├── 副标题: 品牌定位描述
  ├── CTA: "立即了解" → #products / "成为会员" → /register
  └── Hero 图: AI 生成的营养/健康概念图

ProductGallery（第二屏）
  ├── 标题: "十款营养包 · 全人群覆盖"
  ├── 布局: 5列 × 2行 紧凑网格
  ├── 卡片: 渐变色 icon + 产品名 + 一句描述
  └── 交互: 点击弹出详情 Modal（适用人群 + 关键成分标签）

TechSection
  ├── 标题: "技术内核 · 科学驱动"
  ├── 4块: 知识库 · AI模型 · 个性化算法 · 科研文献
  └── 底部: "从数据到方案 · 闭环精准"

KnowledgeBase
  ├── 标题: "知识库 · 科研文献"
  ├── 3列文章卡片（标题 + 分类 + 摘要 + 作者 + 日期）
  └── 底部: 技术白皮书推广区

CTASection
  ├── 渐变色背景（B2B8A3 → D4A373）
  ├── 标题: "加入会员 · 获取专属营养方案"
  └── CTA: "立即成为会员"

Footer
  ├── 品牌 + 3列导航（产品 · 技术 · 关于）
  └── © 2026 + 隐私政策 + 服务条款
```

### 关键设计细节

- 浮动圆动画：Hero 和 Tech 区有 3 个径向渐变圆形，以不同速度浮动（`animate-float-[1-3]`）
- 滚动锚点：`#products`、`#technology`、`#knowledge`
- 卡片进入动画：`framer-motion` + `whileInView` + 交错延迟
- 渐变条：10 款产品的卡片顶部各有不同线性渐变

## 7. 展示图主题框定

### 7.1 展示图生成目的

用于产品介绍、PR 宣传、演示 Deck、App Store / 官网展示等场景。每张图需体现 **精准营养 · 科学驱动** 的品牌内核。

### 7.2 品牌视觉约束

| 维度 | 规范 |
|---|---|
| **品牌色主调** | 暖米色 `#FDFBF7` 背景 + 鼠尾草绿 `#6B705C` CTA |
| **辅助色** | 陶土 `#D4A373` / 苔藓 `#B2B8A3` / 沙色 `#E8D5B7` |
| **字体风格** | 干净无衬线，中文加粗标题，英文标签大写+字距 |
| **圆角语言** | 大圆角卡片（20-32px），胶囊按钮 |
| **阴影感** | 柔和 shadow + 轻微上浮 hover 效果 |
| **氛围** | 自然、健康、温暖、专业 |
| **应避免** | 冷色蓝、荧光色、尖锐直角、杂乱信息 |
